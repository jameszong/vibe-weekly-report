# Weekly Report Generator

A minimal weekly report generator built with Vite + React + Tailwind CSS.

## Features

- **Real-time Preview**: Edit Markdown on the left, see formatted output on the right
- **Minimal Design**: Clean interface inspired by Linear.app
- **Task Lists**: Support for GitHub-style task lists with checkboxes
- **Responsive**: Works on desktop and mobile devices

## Tech Stack

- **Vite**: Fast build tool and dev server
- **React 18**: Modern React with hooks
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **React Markdown**: Markdown rendering with GitHub Flavored Markdown support

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Usage

1. Type your weekly report in Markdown format on the left panel
2. See the formatted preview in real-time on the right panel
3. Use standard Markdown syntax including:
   - Headers (`#`, `##`, `###`)
   - Lists (`-`, `1.`, `2.`)
   - Task lists (`- [x]`, `- [ ]`)
   - Code blocks (```javascript)
   - Inline code (`code`)
